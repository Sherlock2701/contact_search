Please follow this steps to run this program

1)
Run shell script named run_contact.sh
2)
You will see this message on running.
1: Add contact 2:Search 3:Exit

	1:Add contact:- It will add contact to list if its valid contact
	2:Search:- It will search through contact return all contacts with matching contacts
	3:Exit:- It will exit program

3)
You can look up in test_cases.txt file to check test cases you can use for this program.
